## 四、防火墙配置指南
Debian 11系统默认启用ufw防火墙，检测与关闭方法：

bash
# 检查防火墙状态
ufw status

# 若需关闭（适用于无防护需求场景）
systemctl stop ufw
systemctl disable ufw
apt remove --purge ufw
